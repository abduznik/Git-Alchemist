Instant Installation

Linux / Shell

Perfect for Termux, Ubuntu, or WSL.

git clone https://github.com/abduznik/Git-Alchemist.git && cd Git-Alchemist && pip install -r requirements.txt

PowerShell

Optimized for Windows 10/11.

git clone https://github.com/abduznik/Git-Alchemist.git; cd Git-Alchemist; pip install -r requirements.txt

macOS

Native setup for Mac users.

git clone https://github.com/abduznik/Git-Alchemist.git && cd Git-Alchemist && pip3 install -r requirements.txt

Post-Installation Guide

1. Authenticate GitHub CLI Make sure you are logged in to the gh tool: gh auth login
2. Set your API Key Export your Gemini key or add it to a .env file: export GEMINI_API_KEY=your_key
3. Start the Transmutation Run your first optimization: python3 -m src.cli topics
